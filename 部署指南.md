# 在线作业系统 - PythonAnywhere 部署指南

## 第一步：注册 PythonAnywhere

1. 打开 https://www.pythonanywhere.com
2. 点右上角 **Sign Up**
3. 选择 **Beginner account**（免费版）
4. 填写用户名（记住这个，后面要用）、邮箱、密码
5. 确认邮箱

## 第二步：上传代码

### 方式 A：通过网页上传（简单）

1. 登录 PythonAnywhere
2. 点顶部 **Files** 标签
3. 在文件管理器中，创建文件夹 `homework-system`
4. 上传以下文件：
   - `app.py`
   - `wsgi.py`
   - `requirements.txt`
   - 整个 `templates` 文件夹
   - 整个 `static` 文件夹（如果有）

### 方式 B：通过 Git（推荐，后续更新方便）

```bash
# 本地先把代码上传到 GitHub
git init
git add .
git commit -m "initial commit"
git push origin main
```

然后在 PythonAnywhere 控制台：
```bash
git clone https://github.com/你的用户名/homework-system.git
```

## 第三步：修改 wsgi.py

1. 打开 `wsgi.py`
2. 找到这一行：
```python
username = 'ykl'  # 改成你的 PythonAnywhere 用户名
```
3. 把 `ykl` 改成你注册时的用户名

## 第四步：创建 Web App

1. 点顶部 **Web** 标签
2. 点 **Add a new web app**
3. 选择 **Manual configuration**（不要选 Flask 模板）
4. 选择 **Python 3.10**
5. 点 **Next**

## 第五步：配置 WSGI 文件

1. 在 Web 设置页面，找到 **WSGI configuration file**
2. 点链接打开编辑器
3. **删除所有内容**
4. 替换为：
```python
import sys
import os

project_home = '/home/你的用户名/homework-system'
if project_home not in sys.path:
    sys.path.insert(0, project_home)

os.environ['DATABASE_PATH'] = '/home/你的用户名/homework-system/instance/homework.db'

from app import app as application
```
5. 把 `你的用户名` 改成实际用户名
6. 点 **Save**

## 第六步：安装依赖

1. 回到 Web 设置页面
2. 找到 **Virtualenv** 部分
3. 点 **Create a virtualenv**
4. 选择 Python 3.10
5. 创建完成后，打开 **Bash** 控制台
6. 激活虚拟环境：
```bash
workon homework-system
```
7. 进入项目目录：
```bash
cd homework-system
```
8. 安装依赖：
```bash
pip install -r requirements.txt
```

## 第七步：创建数据库目录

在 Bash 控制台运行：
```bash
mkdir -p homework-system/instance
```

## 第八步：启动服务

1. 回到 **Web** 设置页面
2. 点绿色按钮 **Reload ykl.pythonanywhere.com**
3. 等待几秒

## 第九步：测试访问

浏览器打开：`http://ykl.pythonanywhere.com`

应该能看到登录页面！

## 后续更新代码

### 如果用 Git：
```bash
cd homework-system
git pull
```
然后点 Web 页面的 **Reload**

### 如果手动上传：
1. 在 Files 页面重新上传修改的文件
2. 点 **Reload**

## 注意事项

- 免费版每天有一定访问量限制（约 1000 次请求）
- 如果长期不用，Web App 会暂停，访问时点一下 **Reload** 即可恢复
- 数据库在 `instance/homework.db`，所有数据都在里面
- 语音输入需要 HTTPS，PythonAnywhere 免费版是 HTTP，学生可能需要在 Chrome 中手动允许麦克风权限

## 常见问题

### Q: 页面空白/500 错误
A: 查看 **Error log**（Web 页面底部），通常是路径配置问题

### Q: 数据库不工作
A: 确保 `instance` 目录存在，且 `wsgi.py` 中的路径正确

### Q: 修改后没生效
A: 点 **Reload** 按钮
